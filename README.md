# Crud

Projeto em Angular de CRUD com NodeJS

## Iniciar o Projeto

Inicie com o comando `npm install` para instalar todas as dependencias

Depois abra outro terminal e execulte o comando `ng serve` no modo de desenvolvedor e vá para a rota
`http://localhost:4200/`.

## Iniciar o Servidor

Vá para a pasta server com o comando `cd server` e baixe as dependencias com `npm install` 

Logo após crie um arquivo `.env` com as variasveis `DB_USER e DB_PASS` passando os dados do seu mongoDB 

E por fim execulte o comando `npm start`
